CSS Placement

  • External/Linked (外部)
  • Internal/Embedded (內部)
  • Inline (行內/內鍵式)
  • External > Internal > Inline

CSS Placement (contd.)


<!DOCTYPE html>
<html>
  <head>
    <title>我是標題</title>
    <!-- External/Linked CSS -->
    <link rel="stylesheet" href="common.css">
    <!-- Internal/Embedded CSS -->
    <style type="text/css">
      body { background: yellow; }
    </style>
  </head>
  <body>
    <!-- Inline CSS -->
    <h1 style="color:red">哈囉！<h1>
  </body>
</html>

Inheritance

繼承

The process by which properties are passed from parent to child elements even though those properties have not been explicitly defined by other means. Certain properties are inherited automatically, and as the name implies, a child element will take on the characteristics of its parent with regards to these properties.

Inheritance | SitePoint

Specificity

權重/優先級

The concept of specificity states that when two or more declarations that apply to the same element, and set the same property, have the same importance and origin, the declaration with the most specific selector will take precedence.

JS Placement (contd.)


<!DOCTYPE html>
<html>
  <head>
    <title>我是標題</title>
    <!-- External/Linked JS -->
    <script src="common.js"></script>
    <!-- Internal/Embedded JS -->
    <script>
    alert('Hello!');
    </script>
  </head>
  <body>
    <!-- Inline JS -->
    <h1 onclick="alert('Hello Again!');">哈囉！<h1>
  </body>
</html>
